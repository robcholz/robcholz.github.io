# [Foundation] LCEDA

## Quick Action

- Hide Selected Traces: Command+R
- Component Distribute By Region: Shift+P
- Select trace: selected tracing + Shift + W

## Steps

1. Hide all traces.
2. Distribute component by region.
3. Hide all designators.
4. Import DXF board.
5. Start the layout.
